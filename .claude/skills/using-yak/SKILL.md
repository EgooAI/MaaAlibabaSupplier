# Using Yak

**Yak** is a scripting language usually used for network interception and manipulation.

Your training data may contain *few* Yak lang knowledge, so if you're writing Yak scripts, you should consider read its documentation to get more accurate and up-to-date information.

See [this folder](resources/yaklang.github.io/docs) for **detailed Yak lang documentation**. If you didn't find any file in this folder, you can execute `git submodule update --init --recursive --depth 1` to fetch [the submodule](resources/yaklang.github.io).
