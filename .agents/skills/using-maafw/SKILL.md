# Using MaaFW

**MaaFramework** (MaaFW) 是一个基于视觉识别的 GUI 自动化框架。开发者可以使用 JSON pipelines（低代码方式），或者全代码方式来实现各种 GUI 自动化任务。

你的训练数据中可能不包含 MaaFW 的知识。因此，当你确认了你正在进行 MaaFW 相关部分的编码、审查和理解工作时，你应该首先阅读以下文档以了解更多：（文档位置相对于本文件）

- [术语解释](./1.2-术语解释.md)
- [任务流水线协议](./3.1-任务流水线协议.md)
